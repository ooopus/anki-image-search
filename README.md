## Installation

## Usage

1. Open a note in the editor
2. Click the "Search Images" button in the toolbar
3. Select an image from the search results
4. The image will be automatically inserted into the configured field

## Configuration

Access configuration via Tools â†’ Image Search Settings

- **Enabled**: Enable/disable the add-on
- **Search Field**: Field to use as search query
- **Target Field**: Field where images will be inserted
- **Max Results**: Number of search results to display
- **Language**: Interface language

## Requirements

- Anki 25.07 or higher
